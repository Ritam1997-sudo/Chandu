How to Use This Tool

  1. Upload Your Image: Click the upload area or drag and drop your photo or signature file (JPEG or PNG). Only these formats are accepted to ensure compatibility with PAN requirements.
  2. Select Image Type: Choose "Photo" for a 3.5 cm x 2.5 cm image (276x197px at 200 DPI) or "Signature" for a 2 cm x 4.5 cm image (157x354px at 200 DPI) from the dropdown. The width and height inputs will adjust automatically.
  3. Adjust Dimensions (Optional): Modify the width and height if needed. The "Lock Aspect Ratio" checkbox (checked by default) keeps proportions intact. For PAN compliance, stick to the default values unless customizing.
  4. Resize the Image: Click "Resize Image" to process your upload. The tool resizes to the specified dimensions and compresses the output to a JPEG file under 50 KB.
  5. Preview Results: View the original and resized images side-by-side. Check the properties (dimensions, file size) below each preview to ensure they meet PAN criteria (max 50 KB, JPEG, 200 DPI).
  6. Download: If satisfied, click "Download" to save the resized JPEG file, named as "photo-50kb.jpg" or "signature-50kb.jpg" based on your selection.
  7. Reset (Optional): Click "Reset" to clear all inputs, previews, and start over with a new image.
